to run this apps, you'll need :
- `Flutter 3.13.8+`
- `Dart 3.1.4+`

 command to run the apps :
- on browser (chrome)
  ```
  flutter run -d Chrome
  ```
- on ios (simulator)
  ```
  open -a simulator
  flutter run
  ```
